# Smart HR Management System

## Project Overview
A complete HR Management Database System built with MySQL.
Simulates real company HR operations including employee management,
attendance tracking, leave management, and payroll processing.

## Database Schema
- departments — Company departments and heads
- employees — Employee records with department linkage
- attendance — Daily attendance tracking
- leave_requests — Leave application management
- payroll — Monthly salary processing
- hr_audit_log — Automatic audit trail

## Features
✅ Complete Employee Directory View
✅ Department-wise Payroll Reports
✅ Attendance Summary Analysis
✅ Leave Status Tracking
✅ Automatic Audit Logging (Triggers)
✅ Reusable Stored Procedures
✅ HR Dashboard Summary Query

## Tech Stack
- MySQL 8.0
- MySQL Workbench
- Concepts Used: JOINs, Views, Stored Procedures,
  Triggers, Subqueries, CASE WHEN, UNION ALL

## Project Structure
smart_hr_system.sql — Complete database with all tables,
views, procedures, triggers, and sample data
